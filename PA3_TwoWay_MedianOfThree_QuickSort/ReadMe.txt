Tools that used in PA3

IDE:
	Spyder 3.3.6

Programming Language:
	Python 3.7.4

OS:
	Windows 10

Files:
	Cui_PA3.py   	      - This is the source code file which is written in Python 3 on Windows 10. 
		     	      - To execute the program, run the command "python Cui_PA3.py" in the terminal/command prompt
	
	TraceRunReport.txt, Cui_PA3.cover   
			      - These files include the trace run summary which was created by running python built-in trace program.
			      - To reproduce the trace run result, run the command "python -m trace --count --summary Cui_PA3.py"
	
	Cui_PA3_Output.txt
			      - This file includes complete output of both 2-way partition quick sort and median-of-three quick sort.
	

	LongchangCui_PA3_605.621.81.SP21.pdf

			      - This file includes the pseudo code and the answers for each assignment question.
